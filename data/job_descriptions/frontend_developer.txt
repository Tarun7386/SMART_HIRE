Frontend Developer

About the Role:
We are looking for a skilled Frontend Developer to join our team and help create responsive, user-friendly web applications. The ideal candidate will have expertise in modern JavaScript frameworks and CSS.

Requirements:
- Proficiency in HTML5, CSS3, and JavaScript 
- Experience with React.js or NextJS
- Knowledge of responsive design principles
- Familiarity with CSS preprocessors 
- Experience with version control systems (Git)
- Good understanding of browser compatibility issues
- Bachelor's degree in Computer Science or related field (or equivalent experience)

Responsibilities:
- Develop responsive user interfaces for web applications
- Implement UI/UX designs with a focus on usability
- Optimize web pages for maximum speed and scalability
- Collaborate with back-end developers to integrate frontend and backend systems
- Debug and fix issues in frontend applications
- Stay up-to-date with emerging technologies and best practices
